Atom前端插件

Atom插件安装

  1. Emmet — 用过都说好,神器;有个别快捷键会和Markdown preview快捷键冲突,改下就好了
  2. autoprefixer — 用来补充css前缀的,会自动生成多个浏览器的前缀
  3. color-picker — 取色器,太赞了有木有,,比sublime那个好用,不卡,启动超快
  4. atom-html-preview
  5. git-plus，在 Atom 里面执行 Git 命令，不用来回切换终端和编辑器。
